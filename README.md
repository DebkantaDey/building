<h2>📌 Project Overview</h2>

<p>
<strong>BD Brickwork Limited</strong> is a business website designed for a construction and renovation company.
It provides a clean, modern UI and highlights services to help potential clients understand offerings and connect easily.
</p>

<p>
This project demonstrates real-world frontend development skills focused on usability, responsiveness, and business presentation.
</p>

<hr/>

<h2>🚀 Features</h2>

<ul>
  <li>🏠 Modern homepage with company introduction</li>
  <li>🧱 Detailed services section:

  </li>
  <li>📸 Project showcase / gallery</li>
